=== Honeypot Spam Trap ===
Contributors: AScottMcCauley
Tags: Spam
Requires at least: 4.0
Tested up to: 4.1
Stable tag: trunk

Adds several (hidden) decoy fields to the comment form and registration form; Renames the correct fields daily; Automatically stops everyone that fill out those hidden fields.

== Description ==
A basic, low-level trap for comment/login botspam.


== Installation ==

1. Go to your admin area and select Plugins -> Add new from the menu.
2. Search for "SpamTrap".
3. Click install.
4. Click activate.
